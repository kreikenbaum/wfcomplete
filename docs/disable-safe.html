    Firefox: Disable automatic safe mode after crash

    up vote 4 down vote favorite
    2

    How to prevent firefox showing the safe mode dialog after the crash?

    It blocks the automatic selenium tests.

    I have no idea how you got this and what your testing flow is. So I can't reproduce and test the solution. But Firefox Safe Mode can be disable by setting key toolkit.startup.max_resumed_crashes in about:config to -1.

    Here's how to start Firefox with that preference set in C# bidning:

    FirefoxProfile profile = new FirefoxProfile();
profile.SetPreference("toolkit.startup.max_resumed_crashes", "-1");

IWebDriver driver = new FirefoxDriver(profile);
